# Global E-Commerce Transactions Analysis Using Python

# Project Summary:-
This project is an **end-to-end Python Data Analysis project** focused on analyzing **Global E-Commerce Transactions** across **National (India)** and **International Markets**.
Using a structured, multi-notebook approach (Jupter Notebook), the project covers data validation, cleaning, exploratory analysis, customer behavior analysis, sales performance evaluation, and operational efficiency assessment to derive meaningful business insights.

-----

# Dataset Description:
- Total Rows: 201,110
- Total Columns: 15
- Data Scope: National (India) & International Markets
- Domain: Global E-Commerce / Retail
- File Name: "Global_E-Commerce_Transactions.csv"
- Nature: Synthetic dataset designed to mimic real business scenarios.

# Columns Name:
1. Transaction_ID  
2. Transaction_Date  
3. Country  
4. State_Region  
5. Drop_City  
6. Customer_Age  
7. Customer_Gender  
8. Product_Category  
9. Order_Value_USD  
10. Payment_Method  
11. Delivery_Type  
12. Delivery_Days 
13. Order_Status
14. Customer_Rating  
15. Repeat_Customer

-----

# Tools and Techniques Used:-
- Programming Language: **Python**
- Libraries: Pandas, NumPy  
- Data Visualization: Matplotlib, Seaborn  
- Environment: Jupyter Notebook
- Techniques Applied:
  1. Data Cleaning & Preprocessing.
  2. Feature Engineering.
  3. Exploratory Data Analysis (EDA).
  4. Customer Segmentation.
  5. Trend & Pattern Analysis.
  6. Operational Performance Analysis.

# Business Objectives:-
- Analyze global and domestic sales performance.
- Identify high-revenue countries, cities, and product categories.
- Understand customer purchasing and repeat behavior.
- Evaluate delivery efficiency and its impact on customer satisfaction.
- Identify revenue loss due to cancellations and returns.
- Support data-driven business decision-making.
